# XSC 产品定位：学情管家，不是工具

> 核心定位修正：WinGo 学情洞察是**家庭学情管家**，是**家长的教育决策辅助**，不是拍照批改工具。
> 日期：2026-04-21

---

## 一、定位差异

| | 工具思维（错的） | 管家思维（对的） |
|:---|:---|:---|
| **价值感知** | "拍照看对错" | "全年学情追踪 + 决策建议" |
| **使用频率** | 偶尔用（考试前） | 持续用（每周/每月） |
| **家长付费理由** | "分析一次作业" | "请了一个懂教育的管家" |
| **定价锚点** | 按次计费（¥4.9/次） | 按年订阅（¥299/年） |
| **竞品对比** | 和作业帮/小猿比"谁分析得准" | 和家教/培训班比"谁更懂我的孩子" |

## 二、幼升小版：入学管家

**不是"检查孩子会不会数数"，而是"孩子上小学 ready 了吗？"**

### 分析维度
- 数感与计算：10/20以内加减、数量对应
- 识字与阅读：识字量、简单阅读理解
- 专注力与习惯：题目完成度、粗心率
- 逻辑思维：图形推理、规律发现

## 三、小升初版：升学管家

**不是"这道题错了"，而是"孩子升初中的竞争力如何？"**

### 分析维度
- 知识体系完整度：五大模块覆盖情况
- 升学竞争力指数：对标目标初中的要求
- 薄弱点靶向：哪些漏洞会在初中被放大
- 提前衔接建议：哪些初中知识点现在可以渗透

## 四、通用版：学情管家

**跨年级、跨学科的长期追踪与决策支持**

### 核心功能
- 家庭学情档案：多个孩子的学习数据统一管理
- 每学期决策建议：该补哪科、该不该报班、重点投入什么
- 升学节点提醒：幼升小/小升初前自动给出 readiness 评估

## 五、技术实现要点

### Prompt 设计原则
不只是"分析这道题对错"，而是：
1. **诊断**：孩子当前水平在什么位置？
2. **对标**：对标下一个学段的要求，差距多大？
3. **建议**：家长该做什么？不该做什么？
4. **决策**：要不要报班？重点练什么？时间怎么分配？

### 报告模板设计原则
每个报告必须有三个区域：
1. **数据区**：分数、模块强弱、知识点掌握度
2. **建议区**：具体行动清单（本周做什么）
3. **决策区**：家长最关心的问题（要不要报班、重点投什么）

### 年卡核心卖点
- 不是"无限次分析"
- 是"全年管家服务"：每月体检 + 季度复盘 + 年度决策报告
