# WinGo 学情洞察 — 高端幼儿园定价与推广方案

**核心认知**：高端幼儿园家长不是嫌贵，是嫌便宜——低价=低质，高价=专业。

---

## 一、定价策略（高端化）

### 1.1 为什么 ¥299/年太便宜？

| 价格带 | 家长心理 | 结果 |
|--------|---------|------|
| ¥9.9-39 | 便宜货，试试无所谓 | 买了不用，不珍惜 |
| ¥199-299 | 跟普通App一样 | 跟作业帮/小猿搜题比，没差异化 |
| ¥999-1999 | 有点贵，但专业 | "专门为孩子学的，值得" |
| ¥2999-4999 | 高端服务 | "这是高端家庭用的" |

**高端幼儿园家长年消费参考**：
- 幼儿园学费：10-30万/年
- 课外班：3-10万/年
- 游学/夏令营：2-5万/次
- **WinGo 定价 ¥299？不到一顿饭钱，家长会想"这能有用吗？"**

### 1.2 重新定价

#### 方案 A：高端会员制（推荐）

| 档位 | 价格 | 权益 | 定位 |
|------|------|------|------|
| **体验版** | 免费 1 次 | 单次诊断，基础报告 | 引流 |
| **精英版** | ¥999/学期 | 无限次诊断 + 月度学情总结 + 微信推送 | 主流 |
| **尊享版** | ¥2999/年 | 全年无限次 + 成长档案 + 专属学情顾问（线上） + 年度PDF报告 | 高净值 |
| **私享版** | ¥9999/年 | 尊享版全部 + 1对1学情规划师（真人） + 季度家长会PPT定制 + VIP客服通道 | 顶级 |

**关键**：
- 不要卖"工具"，卖"服务"
- 不要卖"次数"，卖"专属"
- 不要卖"AI"，卖"专业"

#### 方案 B：幼儿园打包价（B2B2C）

```
幼儿园统一采购，家长"免费"使用
    ↓
幼儿园向家长收"教育增值服务费"：¥2000-5000/学期
    ↓
其中包含 WinGo 学情诊断 + 其他增值服务
    ↓
幼儿园付给 WinGo：¥500/学生/学期
    ↓
幼儿园毛利：¥1500-4500/学生/学期
```

**家长感知**："幼儿园提供的专业服务"，不是"我买了个App"

## 二、如何推动幼儿园主动推广？

### 2.1 幼儿园为什么要推？

**利益驱动**（最直接）：

| 推广方式 | 幼儿园收益 | 家长感知 |
|----------|-----------|---------|
| 幼儿园统一采购 | 家长交 ¥3000，幼儿园付 ¥500，赚 ¥2500 | "幼儿园服务升级了" |
| 家长自愿购买，幼儿园分成 | 家长付 ¥999，幼儿园分 40%（¥400） | "老师推荐的，试试" |
| 幼儿园收"服务费" | 包装成"教育增值包"，含 WinGo + 其他 | "为了孩子，值得" |

**价值驱动**（长期）：
- 提升幼儿园品牌形象："我们是北京第一家引入 AI 学情诊断的幼儿园"

- 家长满意度："终于知道孩子每天学了什么"

**幼儿园算账**：

- 收入：¥99,900
- 幼儿园分 40%：¥39,960
- 零成本，纯利润

#### 策略 3：帮幼儿园做招生

```
话术：
"X 园长，我们不仅提供学情诊断，还帮您做招生。
· 我们给您的潜在家长免费做学情诊断（引流）

**效果**：幼儿园不仅赚钱，还能招生 → 双赢

#### 策略 4：打造标杆案例

```

## 三、要不要上有赞/微盟等平台？

### 3.1 有赞能做什么？

| 功能 | 适用场景 | WinGo 是否需要 |
|------|---------|---------------|
| 微商城 | 卖商品 | ❌ 不适用 |
| 会员管理 | 用户分层、积分 | ⚠️ 后期可能需要 |
| 分销裂变 | 邀请有礼、拼团 | ✅ 有用 |
| 支付收款 | 微信支付 | ✅ 已有 |
| 数据分析 | 用户行为分析 | ⚠️ 后期可能需要 |
| 小程序商城 | 产品展示+购买 | ⚠️ 可以考虑 |

### 3.2 建议

**第一阶段（现在）**：不需要有赞
- WinGo 是服务，不是商品
- 交易链路简单：扫码 → 授权 → 使用 → 付费
- 有赞反而增加复杂度

**第二阶段（3个月后）**：考虑接入
- 场景：家长"商城"里买会员、买增值服务
- 分销：家长邀请家长，有赞自动结算佣金
- 数据：有赞后台看转化漏斗

**替代方案**：自己开发简单的小程序购买页
```
小程序页面：
├─ 学情诊断（核心功能，免费3次）
├─ 会员中心（购买/续费）
├─ 我的报告（历史记录）
└─ 邀请有礼（分享得奖励）
```

**成本对比**：
- 有赞：¥6800/年 + 交易手续费 0.6%
- 自研小程序：开发成本 ¥5000-10000，后续无年费
