## 二、核心功能验收

### 2.1 OCR 识别
- [ ] 上传清晰作业图片 → 正常识别文字
- [ ] 上传手写/印刷混排 → 正确区分 `[手写]` 标记
- [ ] 低置信度内容 → 标记 `[低置信度]` 并拦截
- [ ] 题目数量异常（<3 或 >30）→ 返回错误提示

### 2.2 AI 学情分析
- [ ] 上传作业 → 生成 `CheckReport` JSON
- [ ] 报告包含：得分、薄弱点、模块得分率、建议
- [ ] 每道题包含：知识点、难度、解析、初中衔接提示

### 2.3 个性化练习生成
- [ ] 根据薄弱点生成 5 道练习题
- [ ] **题型锚定**：与原始试卷题型一致（如看图填词）
- [ ] **难度锚定**：不超过原始试卷难度（基础卷不出拓展题）
- [ ] 每道题包含：题号、题型、内容、答案、解析、难度、知识点

### 2.4 报告展示
- [ ] HTML 报告正常渲染
- [ ] 微信推送通知正常发送

## 三、支付功能验收

### 3.1 免费额度
- [ ] 新用户注册 → 自动获得 3 次免费额度

## 五、部署步骤

1. 确认代码最新：`git pull origin main`
2. Build 镜像：`docker build -t wingo-xsc:latest .`
3. Save 镜像：`docker save wingo-xsc:latest > wingo-xsc.tar`
4. SCP 到 200：`scp wingo-xsc.tar root@101.200.53.200:/opt/`
5. 服务器加载：`docker load < /opt/wingo-xsc.tar`
6. 重启容器：`docker restart wingo-xsc`
7. 健康检查：`curl http://101.200.53.200:3000/api/health`
