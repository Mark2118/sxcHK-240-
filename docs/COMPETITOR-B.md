# B端目标用户深度分析：私立幼儿园 / 托管班 / 私立小学 / 国际小学 / 培训机构

## 一、关键认知转变

**不是和讯飞/好分数/智学网竞争**
- 它们服务公立学校，有GBC渠道壁垒，我们打不进去
- 它们面向K12大规模学校，系统复杂、部署重

**我们要服务的是"小型私立教育机构"**
- 决策快：校长/园长/老板一个人说了算
- 规模小：50-300人
- 付费强：年费¥5000-2万对他们来说是"买个好工具"
- 无政策限制：不需要教育局审批
- 差异化需求强：需要向家长展示"我们用了AI"

---

## 二、各细分用户痛点

### 1. 私立幼儿园

**场景**：
- 家长每月交¥3000-8000学费，期待"高科技感"服务
- 幼小衔接阶段，家长焦虑孩子是否"ready"
- 幼儿园需要生成"成长报告"作为续费依据

**痛点**：
- 老师每天写成长记录耗时耗力
- 家长问"我家孩子今天学了什么"，老师答不上来
- 幼小衔接评估没有数据支撑

**WinGo 价值**：
- 老师上传孩子作业/练习照片 → AI自动生成成长报告
- 报告包含：知识点掌握度 + 薄弱点 + 行动建议
- 自动推送给家长，展示"我们用了AI学情分析"
- **招生卖点**："本园采用AI学情追踪系统，每个孩子都有专属成长档案"

**定价**：¥3000/年/园（100-200个孩子）

---

### 2. 托管班

**场景**：
- 每天下午接孩子 → 辅导作业 → 家长晚上来接
- 家长问"今天作业做得怎么样"，托管班只能答"做完了"
- 竞争激烈，同质化严重

**痛点**：
- 无法向家长证明"我们的辅导是有效的"
- 家长不知道孩子在托管班的真实学习情况
- 续费时没有数据支撑"孩子进步了"

**WinGo 价值**：
- 每天作业拍照 → AI分析 → 生成当日学情简报
- 家长晚上接孩子时，手机上已收到：
  - "今天薄弱点：计算粗心，已练习5道同类题"
  - "本周趋势：比上周错题减少3道 ✅"
- **差异化竞争**：别的托管班说"作业做完了"，WinGo托管班说"今天已针对性巩固薄弱点"
- **续费依据**：月度学情报告，展示"孩子这3个月进步了"

**定价**：¥2000/年/班（20-30个孩子）

---

### 3. 私立小学 / 国际小学

**场景**：
- 班级规模小（20-30人），强调个性化教学
- 家长付费高（¥5-20万/年），期待差异化服务
- 需要"学情档案"作为招生/续费卖点

**痛点**：
- 老师想因材施教，但没有数据支撑
- 家长会问"我家孩子排第几"，学校不便公开排名
- 国际小学需要英文版学情报告

**WinGo 价值**：
- 班级看板：每个学生的薄弱点分布，老师精准调整教学
- 个人档案：每个学生的学情时间线，家长会展示
- 报告自动生成：省去老师手工写评语的时间
- **招生卖点**："本校采用AI学情追踪，每个孩子都有个性化学习方案"
- **国际版**：英文报告，满足国际学校需求

**定价**：¥5000/年/校（10个班级以内）

---

### 4. 培训机构

**场景**：
- 课前：需要诊断学生水平，制定教学计划
- 课后：需要向家长证明"这节课有效"
- 续费：需要数据证明"孩子进步了"

**痛点**：
- 课前诊断靠老师经验，不系统
- 课后反馈靠老师手写，不专业
- 续费时家长说"感觉没进步"，机构拿不出数据

**WinGo 价值**：
- **课前诊断**：学生上传近期作业/试卷 → AI分析薄弱点 → 老师据此备课
- **课后追踪**：每次课后作业拍照 → AI分析 → 家长收到"本次课效果报告"
- **续费依据**：3个月学情对比报告，"计算模块正确率从60%提升到85%"
- **品牌定制**：机构Logo+颜色的报告模板

**定价**：¥2000/年/机构（50-100个学员）

## 四、B/C 双端协同逻辑

### C端家长使用场景

```
孩子在托管班/培训机构上课
     ↓
机构老师用 WinGo B端批量分析全班作业
     ↓
家长收到："XX托管班今日学情简报"
     ↓
家长发现：孩子在机构表现好，但家里作业还有薄弱点
     ↓
家长用 WinGo C端上传家里作业 → 获得个人学情分析 + 行动清单
     ↓
家长看到：机构数据 + 家庭数据合并的完整画像
     ↓
家长决策：知道是否需要额外辅导、报什么班
```

### B端机构获客逻辑

```
C端家长用了 WinGo → 觉得好用
     ↓
家长问机构："你们能不能也用 WinGo 给我们出报告？"
     ↓
机构主动联系 WinGo → 购买 B端服务
     ↓
机构用 WinGo 服务所有家长 → 口碑扩散
     ↓
其他机构看到竞争压力 → 也购买 WinGo
```

**核心逻辑：C端口碑带动B端，B端服务强化C端**

## 五、竞品分析（私立机构市场）

| 竞品 | 目标客户 | 功能 | 定价 | WinGo 优势 |
|------|---------|------|------|-----------|
| **一起作业** | 公立小学 | 作业布置、批改 | 学校付费 | 它们不做私立机构，且功能重 |
| **班级优化大师** | 公立小学 | 班级管理、评语 | 免费+增值 | 不做学情分析 |
| **小打卡** | 培训机构 | 打卡、作业提交 | 免费 | 无AI分析能力 |
| **校宝** | 培训机构 | 教务管理、排课 | ¥3000+/年 | 无学情分析 |
| **小麦助教** | 培训机构 | 教务、家校沟通 | ¥2000+/年 | 无AI学情分析 |

**结论**：私立机构市场的"AI学情分析"是**空白**！
- 校宝/小麦助教做教务管理，不做学情分析
- 一起作业做公立学校，不做私立机构
- WinGo = 第一个面向私立机构的AI学情分析工具
