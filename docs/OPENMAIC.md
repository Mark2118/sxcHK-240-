# OpenMAIC 深度咨询设计文档

## 一、定位

OpenMAIC 是 WinGo 产品生态的**深度咨询层**：
- Dify 处理 80% 简单FAQ
- OpenMAIC 处理 20% 深度咨询（学情解读、学习规划、B端销售）

## 二、C端咨询场景

### 2.1 学情解读

**输入**：用户学情数据（最近5次报告）
**输出**：个性化解读 + 建议

```
家长：孩子计算总是错，是不是该报班？

OpenMAIC：
【现状分析】
- 最近5次作业，计算模块错误率35%
- 主要问题：
  1. 分配律和结合律混淆（出现3次）
  2. 进退位计算粗心（出现2次）

【判断】
这不是"不会"，是"概念不清+习惯问题"

【建议】
1. 先不报班：用WinGo生成的5道针对性练习巩固
2. 观察2周：如果错误率降到15%以下，说明有效
3. 如果持续高于25%：再考虑报计算专题班

【行动】
已为您生成专项练习，今晚开始 → [链接]
```

### 2.2 学习规划

**输入**：孩子当前学情 + 目标
**输出**：个性化学习规划

```
家长：小升初该怎么准备？

OpenMAIC：
【当前水平】计算85%、应用题60%、几何70%
【目标】小升初数学达到90%正确率

【3个月计划】
第1个月：应用题专项（薄弱点TOP1）
第2个月：几何专项（薄弱点TOP2）
第3个月：综合冲刺

【每周安排】
- 周一/三/五：上传作业，WinGo自动分析
- 周二/四：薄弱点专项练习
- 周六：错题回顾
- 周日：休息

【提醒】
需要我每周提醒您执行吗？
```

## 三、B端销售场景

### 3.1 产品介绍

```
机构：你们产品怎么收费？

OpenMAIC：
WinGo 学情引擎为机构提供AI学情分析能力，
不替换现有系统，而是叠加一个学情分析模块。

【定价】
- 托管班：¥2000/年/班
- 培训机构：¥3000-5000/年
- 私立小学：¥5000/年

【核心价值】
1. 向家长证明教学效果（续费率提升）
2. 课前精准诊断（备课效率提升）
3. 品牌定制报告（专业度提升）

需要安排人工详细沟通吗？
```
