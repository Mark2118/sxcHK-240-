## 修复验证

### 修复 1：paperMatch 返回客户端
- **问题**：matchPaper() 被调用但结果未放入 API response
- **修复**：在 `src/app/api/check/route.ts` return 语句中加入 `paperMatch`
- **验证**：API 返回 `paperMatch: {matched: true, paperName: "六年级英语期末卷（看图填词）", confidence: 1}`

### 修复 2：业务校验错误码 500 → 400
- **问题**：题目数量异常（<3 或 >30）和作业内容不完整时返回 500
- **修复**：catch 块中判断错误消息，含"识别结果异常"或"作业内容"时返回 400
- **验证**：题目数量过少/过多均返回 400
